Weather Dashboard
The Weather Dashboard demostrates the use of the OpenWeather API to fetch the current and forecasted weather for the city selected. The last city searched is saved in local storage. CSS is dynamically updated to show weather conditions with representative icons.

Features include:

  • API call to OpenWeather
  • Use of local storage
  • Dynamic update to CSS
